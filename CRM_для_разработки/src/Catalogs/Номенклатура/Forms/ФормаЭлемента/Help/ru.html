Позиция номенклатуры

Справочник Номенклатура предназначен для хранения информации о номенклатурных позициях следующих типов товар, услуга, работа ,тара и набор. Позиции номенклатуры можно дополнительно классифицировать по видам номенклатуры.

Вид номенклатуры – это дополнительная классификация товаров, более обобщенная, чем тип товара. В частном случае вид номенклатуры может совпадать с указанными типами номенклатуры. Однако для более детальной классификации можно использовать несколько видов номенклатуры, имеющих один и тот же тип.

Пример. Если необходимо разделять услуги, которые оказывает наше предприятие, и услуги, оказываемые сторонними организациями, то можно создать два вида номенклатуры: внутренние услуги и сторонние услуги с одним и тем же типом – услуга.

Вид номенклатуры также применяется при ведении учета товаров в разрезе характеристик (цвет, размер, объем морозильной камеры, габариты и т. д.). Для вида номенклатуры можно назначить определенный набор характеристик, который будет общим для всех позиций номенклатуры данного вида.

При описании номенклатурных позиций необходимо определить ряд дополнительных свойств: артикул, внутрифирменное наименование, публичное наименование, изображения.

Кроме того, для позиций номенклатуры определяются ценовая группа, штрихкод и т.д. Для каждой номенклатурной позиции могут быть заданы соответствия позиций номенклатуры позициям номенклатуры партнеров, отличающиеся, например, артикулом или наименованием. Для импортных товаров устанавливается признак ведения учета по ГТД. Информация о номерах ГТД заполняется в документе поставки или в таможенной декларации на импорт.

Информация о новой позиции номенклатуры заполняется с помощью помощника ввода. Список обязательных полей при заполнении и полей для контроля уникальности определяется в справочнике Виды номенклатуры. Для каждого вида номенклатуры можно определить свой список полей, которые будут показываться в помощнике ввода и по которым будет производиться контроль уникальности позиции. Для каждой позиции указывается группа финансового учета номенклатуры.

Товар может приниматься и отгружаться в различных упаковках. При этом для товара можно использовать как наборы упаковок, так и индивидуальные упаковки. Для каждой упаковки может быть назначена своя тара, в которой будет приниматься или отгружаться товар. Если упаковки для товара не используются, товар всегда отгружается в базовых единицах измерения. Информация о таре в этом случае может быть определена для базовой единицы измерения, непосредственно в форме позиции номенклатуры.

Для номенклатуры с типом Набор, в случае если характеристики номенклатуры не используются, доступна команда Состав набора, которая позволяет настроить или изменить состав текущего набора, указать вариант представления набора в печатных формах и правила задания цены набора.

На закладке Драгоценные металлы и камни предусмотрена возможность перечисления драгоценных материалов, входящих в состав номенклатуры. Доступ к этому разделу определяется функциональной опцией Драгоценные металлы и камни в разделе Администрирование-Номенклатура и настройкой учета драгоценных материалов в Виде номенклатуры.

Какую основную информацию необходимо ввести для новой номенклатурной позиции?

В карточку новой позиции номенклатуры вводится следующая информация.

  • Артикул. Артикул позиции номенклатуры.
  • Рабочее наименование. Название товара или услуги, которое используется для быстрой визуальной идентификации номенклатуры в процессе работы с программой.
  • Наименование для печати. Наименование товара и услуги, которое будет печататься во всех документах.
  • Вид номенклатуры. Указывается вид, к которому следует отнести данную позицию номенклатуры. При выборе вида номенклатуры справочно указываются те параметры, которые характеризуют данный вид номенклатуры: тип номенклатуры (товар, услуга, тара, работа), использование характеристик, использование упаковок.
  • Единица хранения. Единица измерения, в которой ведется учет складских остатков товаров и указывается количество номенклатуры в одной упаковке. Для заполнения единицы измерения используется справочник Классификатор единиц измерения.
  • Качество. Можно указать градацию качества вводимой позиции. Для номенклатуры доступно три значения качества: Новый, Ограниченно годен, Не годен. При вводе новой номенклатурной позиции поле Качество по умолчанию заполняется значением Новый. Для каждой степени порчи товара нужно ввести новый элемент справочника Номенклатура. Программа запоминает связь качественного и бракованного товара. Установить связь можно в списке Товары другого качества. Перейти к списку можно с помощью команды панели навигации Подобные товары, далее закладка Товары другого качества.
  • Вести учет по ГТД. Признак того, что для элемента номенклатуры необходимо вести учет по номерам ГТД.
  • Группа номенклатуры. Группа, в которую входит данная позиция номенклатуры в соответствии с классификацией (иерархией) товаров в справочнике Номенклатура.
  • Ценовая группа. Ценовая группа, в которую входит позиция номенклатуры, в соответствии с классификацией товаров по ценовым характеристикам.
  • Драгоценные металлы и камни - перечень входящих в номенклатуру драгоценных материалов, используется в статистических регл. отчетах Форм 1-ДМ, 2-ДМ и 4-ДМ. Для доступа к закладке должна быть включена функциональная опция Администрирование/Номенклатура/Драгоценные металлы и камни и в виде номенклатуры включен учет драгоценных металов и камней.

Как сформировать наименование номенклатурной позиции/характеристики если задан шаблон наименования номенклатуры/характеристики?

Шаблон наименования для позиции номенклатуры и характеристики задается в виде номенклатуры, к которому относится данная позиция. Для каждого шаблона в форме вида номенклатуры можно настроить обязательность его использования с помощью флажков Запретить редактирование.

  • Если флажок установлен, то наименование в карточке номенклатуры или характеристики становится недоступным для редактирования и переформировывается при каждой записи номенклатуры или характеристики.
  • Если флажок не установлен, то заполнить наименование по шаблону можно с помощью команды Заполнить (пиктограмма рядом с полем наименования). При этом если наименование не заполнено, то оно формируется при записи номенклатуры автоматически.

На что влияет установка признака подакцизного товара?

Признак подакцизного товара используется при оформлении розничных продаж, использующих систему налогообложения ЕНВД. Подакцизные товары являются объектами налогообложения в любом случае, независимо от того, применяется ли система налогообложения ЕНВД при розничной торговле. Список подакцизных товаров регламентирован в соответствии со статьей 181 НК РФ.

  • При оформлении продаж с розничного склада, применяющего систему ЕНВД, обобщенному покупателю (документы Чек ККМ, Отчет о розничных продажах) для подакцизных товаров устанавливается ставка НДС, указанная в форме номенклатуры.
  • Подакцизные товары запрещено продавать в рамках соглашений (документов) с видом налогообложения Продажа облагается ЕНВД.

Как ввести дополнительные сведения для позиции номенклатуры?
  • Выберите Все действия-Дополнительные сведения. Можно также воспользоваться соответствующей пиктограммой.
  • Откроется список дополнительных сведений.
  • В списке представлены те дополнительные сведения, которые являются общими или введены для вида номенклатуры, к которому относится данная позиция.
  • Введите значения этих дополнительных сведений. В качестве значений могут использоваться различные виды значений: число, дата, текстовая строка, различные справочники. Это определяется тем какой тип значения задан для дополнительного свойства в наборе свойств для данного вида номенклатуры.

Если планируется использовать конкретную позицию номенклатуры в качестве комплекта, то как можно указать комплектующие (спецификацию комплекта)?
  • Выберите пункт меню Варианты комплектации в панели навигации формы.
  • Создайте вариант комплектации, в котором укажите список комплектующих.
  • Можно создать несколько вариантов комплектации, один из вариантов можно отметить, как основной.

Операция по сборке комплектов оформляется документом Сборка(разборка) товаров. Предусмотрена возможность оформления предварительного заказа на сборку(разборку) товаров. При заполнении комплектующих в документах по умолчанию используется основной вариант комплектации.

Важно. Комплектом может быть только та позиция номенклатуры, которая является товаром (относится к виду номенклатуры, в котором определен тип номенклатуры -товар).

Как посмотреть место размещения товара на складе?

Информацию о том, где находится конкретный товар можно посмотреть, используя в навигационной панели пункт Размещение в ячейках . Информация о размещении товара на конкретном складе, в конкретном помещении и конкретной ячейке задается в момент приема товара на склад кладовщиком.

Важно. Информация о размещении доступна только в том случае, еслт товар размещен в ячейке адресного склада (помещения). Для склада (помещения), на котором хранится товар, должен быть выбран вариант работы с ячейками Использовать для хранения остатков номенклатуры.

Как посмотреть информацию по ценам номенклатуры?

Из формы номенклатуры можно посмотреть дополнительную информацию о ценах, которые назначены данной позиции номенклатуры. Для просмотра используется пункт навигационной панели Цены. Дополнительная информация показана на закладках.

  • Прайс-лист. Здесь можно увидеть цены для текущей позиции номенклатуры в виде настраиваемого отчета.
  • Цены номенклатуры. Показывается список цен, зарегистрированных для данной позиции номенклатуры, документом Установка цен номенклатуры.
  • Цены номенклатуры поставщиков. Список цен поставщиков, зарегистрированных для данной позиции номенклатуры. Информация об этих ценах регистрируется при оформлении документа Регистрация цен поставщика.

Где можно посмотреть и отредактировать параметры управления запасами, относящиеся к данной номенклатуре?

Из формы номенклатуры можно перейти к информации о параметрах управления запасами для данной номенклатуры. Для этого в навигационной панели выберите пункт Параметры обеспечения потребностей. Далее выберите одну из закладок.

  • Параметры управления запасами
  • Настройка контроля остатков
  • Настройка передачи материалов в производство
